# AI-Cybersecurity Cloud Protection

## Project Overview
This project integrates Artificial Intelligence (AI) with cybersecurity to detect and prevent cyber threats in cloud-based environments. It leverages machine learning models to analyze network traffic, identify anomalies, and provide proactive threat mitigation solutions. The project is aimed at improving cybersecurity defense systems in real-time.

## Tools & Technologies
- **Programming Languages**: Python, Bash
- **Machine Learning Tools**: Scikit-learn, TensorFlow
- **Network Tools**: Wireshark, Nmap, Snort IDS
- **Cloud Platforms**: AWS, Google Cloud Platform
- **Data Visualization**: Grafana, Power BI

## Setup & Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/abdulrahman-haggam/AI-Cybersecurity-Cloud-Protection.git
    cd AI-Cybersecurity-Cloud-Protection
    ```

2. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the script:
    ```bash
    python run.py
    ```
